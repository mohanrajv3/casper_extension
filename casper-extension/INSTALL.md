# Install CASPER Extension

## Chrome / Edge (Unpacked)
1. Open `chrome://extensions` (or `edge://extensions`)
2. Enable Developer mode
3. Click `Load unpacked`
4. Select folder: `/Users/mohan/casper_web_extension/casper-extension`
5. Pin the extension from the toolbar puzzle icon

## First setup
1. Open popup
2. Create PIN (4-6 digits)
3. Optionally set alert email (recommended)
4. Unlock and start using vault

## Quick verification
1. Add a password manually in popup or vault
2. Visit same site and test autofill
3. In vault `Settings`, set alert email and click `Send Test Email`
4. In vault `Security`, click `Trigger Breach Test`
5. Check inbox/spam for welcome and breach emails
